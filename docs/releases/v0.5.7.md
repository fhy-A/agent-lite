# Code v0.5.7 Release Notes

Date: 2026-07-23

Code v0.5.7 focuses on completing the Skill dependency lifecycle in settings:

- Runtime-only dependency installation/repair/uninstall (Python & Node) now has user-visible actions and status feedback.
- System-level dependencies remain read-only hints (`installHint`) and are no longer installed automatically by the model.
- Added `Install Missing Required Dependencies` at skill level for one-click batched handling.
- Skips optional dependencies by default (unless manually requested), avoiding unintended environment changes.
- Improved status consistency after install/reinstall/check operations.
